# 法护星 - 专业法律助手

智能法律咨询服务平台

## 功能特性

- ⚖️ 法律智能咨询 - AI驱动的法律问题解答
- 📄 合同风险审查 - OCR识别 + AI分析
- 📝 文书生成 - 自动填充起诉状模板

## 快速开始

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器
npm run dev
```

## 环境变量

复制 `.env.example` 到 `.env.local` 并配置：

```
YUANQI_API_KEY=your_api_key
YUANQI_ASSISTANT_ID=your_appid
API_BASE_URL=http://localhost:8000
```
